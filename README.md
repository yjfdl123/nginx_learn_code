# nginx_learn_code

生成多个配置的nginx配置，用于学习


### 01_simple 建立主页

01_simple    主要是把nginx搭建起来，能够看到index.html页面

1. 一定要把各种 include *.conf 去掉，不然会被冲淡
2. 要设定好目录，html文件夹要有目录，不然就会报错，
3. 使用root问题就简单了

### 02_域名注意

当没有绑定域名的时候，可以通过ip直接访问80端口
但是当绑定了域名的时候，80端口只能通过域名来访问了
